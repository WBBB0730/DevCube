# JetBrains Mono（完整可变字体）

来源：[JetBrains/JetBrainsMono](https://github.com/JetBrains/JetBrainsMono) **v2.304** 官方 release（`fonts/variable/*.ttf` → 本地转 woff2）。

不用 `@fontsource-variable/jetbrains-mono`：其按 script 切的子集**不含** `⇧⌃⌥⌘` 等修饰键字形；完整面才有（与 WebStorm 自带 Mono 一致）。

许可见同目录 `OFL.txt` / `AUTHORS.txt`。
